# 13. Templates & Artifacts

Reading about the craft is half the job. Executing it under deal pressure requires artifacts you can open, fill in, and send **today** — without reinventing the structure every time.

This section contains **twelve copy-paste templates**, one for every stage of the enterprise deal. Each template is:

- **Stage-tagged** — you always know *when* in the deal it applies.
- **Plain markdown** — paste it into your notes tool, wiki, doc editor, or CRM. No proprietary formats.
- **Time-boxed** — every artifact is designed to be completed in under 45 minutes.
- **Cross-referenced** — each links back to the playbook chapters that teach the thinking behind it.

> **How to use this library:** Don't fill in templates for their own sake. Each artifact exists to force a *decision* — qualify or walk, proceed or stop, win declared or loss called. If completing a template doesn't change what you do next, you didn't need it for that deal.
